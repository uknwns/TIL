## 재귀함수

재귀 함수란 임의 함수 test에서
test함수를 호출 하는 것과같이 임의 함수에서 같은 함수를 호출 하는 것과 같다.

```js
예시)
function test () {
    return test()
}
```
하지만 위 함수처럼 작성하게 되면 영원히 끝나지 않는 함수가 되어서 무한 루프가 되어서
오류가 발생한다.

그래서 

```js
예시)
function test (cnt = 0) {

    if(cnt === 10) { 
    
    return;

    }

    return test(cnt + 1)
}
```

위 예시와 같이 매개변수와 탈출조건이 동반해야 무한 루프가 되지 않고 원하는 방향으로
코드를 작성할 수가 있다.
